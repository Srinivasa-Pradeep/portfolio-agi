# Srinivasa Pradeep - Technical Portfolio

An architecturally refined digital environment and professional showcase engineered with Next.js 15, Firebase, and Google Genkit. This platform serves as a high-performance interface for demonstrating expertise in software engineering, technical writing, and systems design.

## Core Systems

### Liz - Neural Concierge
A sophisticated AI assistant powered by Gemini 3.1 Flash-Lite. Liz is engineered to represent professional milestones, research findings, and core philosophies with architectural precision. The system features a continuous acoustic awareness engine designed for seamless interaction.

### The Horizontal Odyssey
A cinematic, F1-inspired career timeline utilizing liquid-glass telemetry interfaces. It features physics-based movement and an atmospheric star-field engine optimized for GPU acceleration.

### Zen Mode
A focus-engineered productivity environment featuring a luminance-controlled celestial interface. This module utilizes specialized React hooks to manage screen wake-locks and session-based growth algorithms.

### Technical HUD
Real-time synchronization with LeetCode and GitHub APIs. This dashboard provides live telemetry on contribution velocity and competitive programming metrics using high-fidelity data visualization components.

### Personal Library
A reorderable, 3D-modeled rack of influential volumes. The interface prioritizes tactile feedback and clean typography, categorized by Logic, Growth, and Imagination.

## Technology Stack

### Frontend Architecture
- Core Framework: Next.js 15 (App Router)
- Component Library: Shadcn UI / Radix UI
- Styling Engine: Tailwind CSS
- Animation: Framer Motion / Three.js
- State Management: React Context / Custom Hooks

### Intelligence Layer
- AI Framework: Google Genkit
- Primary Model: Gemini 3.1 Flash-Lite
- Audio Processing: Web Speech API / MediaStream Hardware Lock

### Infrastructure and Persistence
- Backend: Firebase (Firestore, Authentication, App Hosting)
- Deployment: Firebase App Hosting
- Data Fetching: Server Actions / SWR-style polling

## System Structure

```
├── src/
│   ├── ai/            # Genkit flows and neural configurations
│   ├── app/           # App Router segments and Server Actions
│   ├── components/    # Atomic UI units and modular systems
│   ├── context/       # Global state management
│   ├── firebase/      # Client-side SDK and database hooks
│   └── lib/           # Static data and utility functions
```

## Development and Deployment

1. Configuration: Populate environment variables in .env with GEMINI_API_KEY and Firebase credentials.
2. Installation: Execute `npm install` to initialize dependencies.
3. Execution: Run `npm run dev` to start the high-performance development server.
4. AI Development: Utilize `npm run genkit:dev` for prompt engineering and flow testing.
