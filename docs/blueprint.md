# **App Name**: Srini Portfolio

## Core Features:

- Hero Section: Display name, title, and a one-liner, with buttons to view projects and download resume.
- About Section: Show a short bio, quick stats, and a skills grid with category tags.
- Blogs Section: Display a grid of blog cards with title, summary, date, read time, and tags. Include a button to view all blogs.
- LeetCode Section: Present LeetCode stats (total solved, easy/medium/hard, streak, contest rating), a progress chart/heatmap, and a list of featured solutions with links.
- Projects Section: Display featured project cards with project name, problem solved, tech stack, and links to GitHub and live demo.
- Contact Section: Include a contact form, email link, and links to LinkedIn and GitHub. A LLM is used as a tool that analyses and classify the content of the email and assign appropriate labels and categories based on context. Also includes minimal footer with copyright info.
- Theme Switcher: Allow users to switch between light and dark mode with a toggle.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke professionalism and calm confidence.
- Background color: Light gray (#F0F0F0) for light mode, dark gray (#333333) for dark mode.  Both desaturated and of the same hue as indigo.
- Accent color: Soft lavender (#E6E6FA), approximately 30 degrees to the 'left' of indigo on the color wheel, and sufficiently distinct in brightness and saturation for good contrast.
- Headline font: 'Playfair', serif.  Body font: 'PT Sans', sans-serif.  Pairing chosen for elegant, readable experience.
- Spacious layout with large whitespace, soft shadows, and rounded corners for a modern and premium look.
- Smooth fade-in on scroll for each section, button hover with slight lift and glow, cards with soft scale and shadow on hover. Navbar blur effect on scroll.
- Simple, minimalist icons for navigation and links.
